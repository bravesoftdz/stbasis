Adapters Example

This example provides comprehensive testing of Data Abstract adapters.

There are two main functions and these are available via the Test Datasets and Test Deltas buttons.

You must click Test Datasets first in order to open the master/detail tables. Next, modify data in one or both tables and then try Test Deltas and/or Show Delta Counters.

See the affect of changing the check box settings.

It is instructive to examine the source code within fMainForm to see how this functionality has been achieved.
