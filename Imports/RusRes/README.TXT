В этом архиве предлагается Вашему вниманию набор руссифицированных ресурсов
для Delphi 3, 4 и 5. (только для RUNTIME при откл. Build with runtime packages)

Версия 2.1

Для использования необходимо:

Для Delphi 3,4,5 предлагается частичная руссификация runtime. Переведены строковые ресурсы,
находящиеся в представленных PAS-файлах. В соответствии с рекомендациями Inprise Corp.
необходимо записать их в папку, доступную для Ваших проектов (либо туда, где сам проект, либо вписать в search path в опциях проекта).

Примечание:

- файлы sysconst.pas, corbcnst.pas, comconst.pas, midconst.pas, scktcnst.pas - только для Delphi 4 и 5, 
  для Delphi 3 - sysutils.inc (include-файл для sysutils.pas).
- qr3const.pas - для Quick Report 3.0.5.
- файлы adoconst.pas, brkrconst.pas, ib.pas, mxdconst.pas, wbmconst.pas - только для Delphi 5.

Ждем Ваших предложений и пожеланий.
